**git 반영하기

작업하기 => (선택) git status : 어떤 파일이 바뀌었는지 확인 가능

=> git add . : 모든 경로에 바뀐 파일 적용

=> git commit -m (커밋메시지) : ex) git commit -m "추가: 전투 애니메이션"

=> git push (서버주소) (올릴 브랜치) : ex) git push origin develop 



**git 리포지토리 서버 주소 확인하는법

git remote -v



**리포지토리 서버 별명 추가
